nt_rms

PURPOSE

RMS - Root-mean-square

SYNOPSIS

function y=rms(x, dim)

DESCRIPTION 

RMS - Root-mean-square
 
  y=rms(x): If x is a matrix the rms is calculated columnwise.
  y=rms(x,dim): calculate rms according to dim.
 
  see also nt_bsrms, nt_bsplot
